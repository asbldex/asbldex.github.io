Automated Synthetic Biology Labs

ASBL makes machines for keyboard biotechnology. Ditch the gloves, be a keyboard bioengineer.

ASBL is a biotechnology company developing machines for bioengineers. Our machines are ASBL Lab 0, ASBL 1 and ASBL II. Biotechnology should be accessible to everyone. An ASBL machine is an Automated Synthetic Biology Lab. ASBL simplifies biotechnology so anyone can do it without an expensive lab or PhD training. ASBL Machine Protocol is our language of biotechnology, like Assembly. ASBL Machine Protocol is the operating language of ASBL.

Biotechnology research protocols are written in research methods today (Bio-Protocol). ASBL Machine Protocol helps decompose research protocols into unit operations on ASBL machines. Decomposing a protocol into a series of steps, and composing an ASBL Machine Protocol file with these steps translates the protocol to ASBL. This file is transmitted over the Internet to replicate the protocol on other ASBL machines.

ASBL is developing RNA-based drugs for nonalcoholic fatty liver and type 2 diabetes to benchmark ASBL machines in drug development. ASBL's cutting-edge biotechnology tools include Deconstructed Human organoid chips, high-throughput assays and targeted delivery. Our drugs bioengineer by harnessing the human body's innate mechanisms to target inflammation and glucose imbalance.

We build and freely release computational biology software:
1. HumanGEMlib Python library
2. Bioreactor Model Python library
3. Protocol Dex biotechnology protocol index

ASBL Inc. is incorporated in Delaware, United States and headquartered in Boston, Massachusetts, United States.
